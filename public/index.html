# API Functions API Requirements API Route
User Routes
1 Sign Up (post) User Data /signup
2 Login (post) username, password /login
3 User Profile (get) token /profile
4 User Logout (get) Nothing /logout
5 Particular User (get) User ID /user/:id
Post Routes
1 Load Posts (get) Nothing /posts
2 Post by Id (get) Post ID /post/:id
3 Add a question (post) Token /addQuestion
Answer Routes
1 Add answer (post) Token, Post Id asdf
2 Load Answer (get) Post Id /answers/:id
Comment Routes
1 Add Comment (post) Token, Post Id, Answer Id /addComment
2 Load Comment (get) Post Id /comment/:id
Vote Routes
1 Upvote (post) Token, Post Id, Answer Id /upvote
2 Downvote (post) Token, Post Id, Answer Id /downvote
3 Unvote (post) Token, Post Id, Answer Id /comment/:id
Subject Routes
1 Add Subject (post) Name, description, picture /subject
2 Load Subject (get) Nothing /subject
3 Load a picture (get) pictureName /picture/:pictureName
Topic Routes
1 Add Topic (post) Subject Id; Name, description, picture /topic
2 Load Subject (get) Subject Id /topic
3 Load a picture (get) pictureName /topic/:pictureName
Chapter Routes
1 Add chapter (post) Subject Id, Topic Id; Name, content, picture /chapter
2 Load chapter (get) Subject Id /chapter
3 Load a picture (get) pictureName /chapter/:pictureName
